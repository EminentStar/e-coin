# e-coin

Hypothetical e-coin system developed by Junkyu Park. In this system, people can sign un/in as a user in the system, and they can exchange, refund, remit e-coin, and buy product on the Internet.

### Requirements

- python(3.5.2)
- django(1.8)
- psycopg2
- BeautifulSoup4

### Features

- Sign up/in
- Exchange e-coin
- Buy products
- Refund e-coin
- Remit e-coin
